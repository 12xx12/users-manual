This is the Cuberite User's Manual. It is intended as a guide on how to operate the basics of Cuberite, as well as how to use some of the more advanced features. It is organised into major sections as well as more detailed subsections and these are laid out in the {{toc - Table of Contents}}.

As with Cuberite itself, this manual is in active development, and you can contribute yourself or file an issue if it's out of date on the GitHub Repository.
