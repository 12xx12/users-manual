MCServer is a Free and Open Source (FOSS) Minecraft Server. MCServer has been designed to keep in consideration performance, configurability and extensibility, as well as to replicate many vanilla features. MCServer is written in C++, although there is an expansive plugin system utilising Lua. In fact, many of the built in commands are implemented by the Core plugin, which has it's own GitHub repository and developer community.

MCServer today is maintained by a large team of open-source contributors and plugin developers. If you want to reward the developers for their work, you can set up a donation on Gittip or Tip4Commit (using bitcoin).
