Cuberite is a Free and Open Source (FOSS) Minecraft-compatible Game Server. Cuberite is designed with performance, configurability, and extensibility in mind, and also aims to accurately recreate most vanilla features. Cuberite is written in C++, and there is also an expansive plugin system that allows for the user to write their own plugins with Lua. In fact, many of the built in commands are implemented by the Core plugin, which has it's own GitHub repository and developer community. For more information on the plugin system and how to use it, as well as how to develop for it, please see section 4.1 {{4.1 - Introduction to Plugins}}.

Cuberite today is maintained by a large team of open-source contributors and plugin developers. If you want to reward the developers for their work, you can set up a donation on Bountysource (with a credit card, bitcoin or other currency).
