You can link worlds without resorting to plugins by modifying world.ini files. However, this method is limited: Each world can only be linked to 2 different worlds.

By default, stepping through any Nether Portal leads to the Nether, and stepping through any End Portal leads to the End. This behavior can be tweaked, and you can make these portals teleport to a world of your choice. This is done by editing the [LinkedWorlds] section of each world's world.ini file. Note that with this method, you can't make two different portals of the same type teleport you to two different worlds. If you want such behavior, use a plugin and read the previous subsection.
