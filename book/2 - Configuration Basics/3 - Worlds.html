By default, there is only one world. That world can be tweaked by editing the file world/world.ini. You can edit things like the spawn point location and the game mode. See section 3 - {{3 - Configuring world.ini}} for details.

If you have the permission, you may use the following command to travel between worlds: /portal Worldname
