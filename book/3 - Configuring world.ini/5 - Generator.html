This section specifies all the options for Cuberite's world generator.

World Generators

Cuberite allows multiple world generators. Currently there is only one, the ComposableGenerator.

  • ComposableGenerator: The ComposableGenerator allows an admin to mix and match multiple different biome, shape, composition and finishers with each other allowing great freedom in what you let the generator create.

    OptionsDescription
    BiomeGen The biome generator used by the ComposableGenerator
    ShapeGen The shape generator used by the ComposableGenerator
    CompositionGen The shape generator used by the ComposableGenerator
    Finishers All the structures, foliage and other things that come after composition

Biome Generators

Cuberite has multiple biome generators. Some are for developers to make their life easier. To specify the biome generator you want to use you set the BiomeGen value to one of the available options.

Constant

Allow only one biome in the world, the one you specify. This is usefull if you want a superflat world

OptionsDescription
ConstantBiome The name of the biome you want to use. For instance, Forest or SavannaPlateauM

Checkerboard

Generates specified biomes in a checkerboard shape. This is intended for developers.

OptionsDescription
CheckerBoardBiomes The biomes you want to generate separated by a comma.
CheckerboardBiomeSize The size of each square.

Voronoi

Generates biomes based on a voronoi diagram. The borders from this generator are straight which can be really notable. Furthermore unrelated biomes can easily come next to each other (Taiga next to a jungle). This generator is outdated and not recommended for general use.

OptionsDescription
VoronoiCellSize Roughly the size of each biome
VoronoiJitterSize TODO
VoronoiOddRowOffset TODO
VoronoiBiomes The biomes you want to generate separated by a comma.

DistortedVoronoi

Generates biomes based on a voronoi diagram. Unlike the Voronoi generator the borders look more natural. Just like with Voronoi unrelated biomes can easily come next to each other (Taiga next to a jungle). This generator is outdated and not recommended for general use.

OptionsDescription
DistortedVoronoiCellSize Roughly the size of each biome
DistortedVoronoiBiomes The biomes you want to generate separated by a comma.

MultiStepMap

MultiStepMapRiverCellSizeRoughly the size of rivers.
MultiStepMapLandBiomeSizeRoughly the size of land.

Grown

Cuberite's latest, fastest and most complete 1.7 biome generator.

This biome generator currently does not have any options you can customize.

GrownProt

Identical to Grown, but aimed at developers. It's a little bit slower, but the source is more readable.

This biome generator currently does not have any options you can customize.
